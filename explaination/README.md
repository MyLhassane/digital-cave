# مشروع Digital Cave (البيئة الكريستالية)

## نظرة عامة

مشروع **Digital Cave** هو تجربة تفاعلية ثلاثية الأبعاد (3D) مبنية باستخدام مكتبة **Three.js**، تهدف إلى تقديم المعرفة بطريقة بصرية immersive. يشبه المستخدم أنه يطير داخل كهف بلوري (كريستالي) عملاق، وكلما تعمق أكثر يواجه فروعًا علمية مختلفة يمكنه النقر عليها لاستكشاف المزيد.

---

## هيكل المشروع

```
digital-cave/
├── index.html              # صفحة المدخل الرئيسي — يستورد main.js
├── sections.html           # صفحة الأقسام الفرعية — يستورد sections-main.js
├── posts.html              # صفحة المقالات — يستورد posts-main.js
├── styles/
│   └── main.css            # الأنماط والتنسيقات العامة
├── js/
│   ├── config.js           # إعدادات الفروع وحدود الحركة
│   ├── geometry.js         # دوال إنشاء الهندسة الكريستالية
│   ├── scene.js            # إنشاء المشهد والكاميرا والإضاءة
│   ├── animate.js          # حلقة التحريك (مع onFrameUpdate)
│   ├── controls.js         # التحكم في الحركة والاختيار (مع onExceedMax)
│   ├── ui.js               # عناصر واجهة المستخدم (اللوحات)
│   ├── main.js             # [صفحة المدخل] ربط المكونات للفروع الثلاثة
│   ├── sections-main.js    # [صفحة الأقسام] محتوى الأقسام الفرعية
│   └── posts-main.js       # [صفحة المقالات] محتوى المقالات + تأثيرات خاصة
└── explaination/
    └── ...                 # هذا المجلد - شروحات لكل ملف
```

---

## تدفق العمل (User Flow)

```
index.html  (المدخل — 3 فروع)
    │
    ▼ النقر على فرع
sections.html  (الأقسام الفرعية حسب المجال)
    │
    ▼ النقر على قسم
posts.html  (10 لوحات مقالية)
```

---

## البنية المعيارية (Modular Architecture)

جميع الصفحات الثلاث تشترك في 5 وحدات أساسية:

```
           ┌─────────────┐
           │   config.js  │  إعدادات الفروع
           └──────┬──────┘
                  │
┌─────────┬───────────────┬──────────┬──────────┐
│scene.js │ geometry.js   │ animate  │controls │
│(مشهد)   │ (هندسة كهف)   │ (تحريك)  │(تحكم)   │
└────┬────┴───────┬───────┴────┬─────┴────┬─────┘
     │            │            │          │
     ▼            ▼            ▼          ▼
┌────────────────────────────────────────────────┐
│            main.js  (index.html)                │
│         sections-main.js  (sections.html)       │
│           posts-main.js  (posts.html)           │
└────────────────────────────────────────────────┘
```

**لا يوجد كود مكرر.** كل صفحة تستورد فقط ما تحتاجه من الوحدات المشتركة.

---

## التقنيات المستخدمة

- **Three.js r128** — مكتبة الرسم ثلاثي الأبعاد
- **JavaScript ES Modules** — تنظيم الكود في وحدات مستقلة
- **Canvas 2D API** — رسم النصوص والأنماط على اللوحات
- **CSS3** — التنسيقات البصرية والانتقالات

---

## كيفية التشغيل

المشروع يعمل بالكامل في المتصفح. افتح `index.html` في أي متصفح حديث. يُفضّل استخدام خادم محلي بسيط:

```bash
# باستخدام Python
python3 -m http.server 8000
# ثم افتح http://localhost:8000
```
