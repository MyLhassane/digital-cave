# js/config.js — إعدادات الفروع وحدود الحركة

## ما هو هذا الملف؟

هذا ملف تهيئة (Configuration) يصدّر ثوابت تُستخدم في جميع أنحاء المشروع. يساعد على تجنب الأرقام السحرية (magic numbers) وتوحيد القيم بين الملفات.

---

## المحتوى

### 1. تكوين الفروع (branchesConfig)

```javascript
export const branchesConfig = [
    {
        id: "natural",
        name: "العلوم الطبيعية",
        zFork: -600,           // العمق الذي يظهر فيه الفرع
        angle: -Math.PI / 4,   // زاوية الفرع (-45 درجة)
        side: -1,              // الجهة (يمين/يسار)
        color: "#00d4ff"       // لون التوهج السماوي
    },
    {
        id: "social",
        name: "العلوم الاجتماعية",
        zFork: -1400,
        angle: Math.PI / 4,    // +45 درجة
        side: 1,
        color: "#9d00ff"       // لون توهج أرجواني
    },
    {
        id: "formal",
        name: "العلوم الشكلية",
        zFork: -2200,
        angle: -Math.PI / 4,   // -45 درجة
        side: -1,
        color: "#00ff87"       // لون توهج أخضر
    }
];
```

كل فرع يمثل مجالًا معرفيًا رئيسيًا وله:
- **zFork**: الموقع على عمق المحور Z حيث يظهر الفرع (الأرقام السالبة تعني "أمام" الكاميرا).
- **angle**: الزاوية التي ينحني بها الفرع (يمينًا أو يسارًا).
- **side**: الاتجاه الجانبي (+1 يمين، -1 يسار).
- **color**: اللون المميز للفرع.

### 2. حدود الحركة (movementBounds)

```javascript
export const movementBounds = {
    minZ: -2900,   // أقصى عمق يمكن للمستخدم الوصول إليه
    maxZ: 150      // أقصى ارتفاع (العودة للمدخل)
};
```

تمنع الكاميرا من تجاوز حدود الكهف، وتحافظ على تجربة سلسة بدون أخطاء بصرية.

---

## لماذا هذا الملف مهم؟

- **مركزية الإعدادات**: تغيير موقع فرع أو لونه يتم من مكان واحد.
- **قابلية التوسع**: يمكن إضافة فروع جديدة بسهولة بإضافة عنصر جديد للمصفوفة.
- **وضوح الكود**: الأرقام لها أسماء معبرة بدلًا من أرقام عشوائية في الكود.
