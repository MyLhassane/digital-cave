# js/ui.js — عناصر واجهة المستخدم (اللوحات الإرشادية)

## ما هو هذا الملف؟

هذا الملف مسؤول عن إنشاء **العناصر البصرية التفاعلية** التي يراها المستخدم في الكهف، وهي اللوحات الإرشادية (signboards) المتوهجة التي تشير إلى فروع المعرفة المختلفة. كما يدير تأثيرات الانتقال بين الصفحات.

---

## الدوال

### 1. `createGlowingSignboard(text, colorHex)`

تنشئ **نقشًا (Texture)** بلوحة إرشادية متوهجة باستخدام Canvas 2D:

#### خطوات الرسم على Canvas:
1. **خلفية داكنة**: مستطيل داكن (`rgba(3, 9, 20, 0.9)`) بمقياس 512×180 بكسل.
2. **إطار متوهج**: مستطيل بحدود بلون الفروع (`colorHex`) وعرض 8 بكسل.
3. **تعبئة متدرجة**: تدرج لوني من الأزرق الداكن (`#011f38`) إلى لون الفرع.
4. **النص**: النص العربي بخط عريض 42 بكسل في المنتصف.

النتيجة: **نقش (CanvasTexture)** جاهز للاستخدام كمادة (material) في Three.js.

### 2. `createBranchSignMesh(cfg)`

تنشئ **مجسم اللوحة** في الفضاء ثلاثي الأبعاد:

```javascript
const signMesh = new THREE.Mesh(new THREE.PlaneGeometry(160, 55), signMat);
```
- شكل مستطيل (عرض 160، ارتفاع 55 وحدة ثلاثية الأبعاد).
- المادة (material) من نوع `MeshBasicMaterial` (لا تتأثر بالإضاءة، تبقى ساطعة دائمًا).

**تحديد الموقع:**
```javascript
signMesh.position.set(0, 0, cfg.zFork);
signMesh.rotation.y = cfg.angle * 0.5;  // نصف زاوية الفرع
signMesh.translateZ(-300);  // أمام الفرع بقليل
signMesh.translateX(cfg.side * -50);  // منحرفة قليلاً للجانب
signMesh.position.y = 80;  // مرتفعة قليلاً للأعلى
```

**بيانات المستخدم (userData):**
```javascript
signMesh.userData = { redirectUrl: `sections.html?domain=${cfg.id}` };
```
تخزن الرابط الذي سينتقل إليه المستخدم عند النقر على اللوحة. هذا ما يستخدمه Raycaster في `controls.js`.

### 3. `startFadeAndRedirect(url, delayMs = 600)`

تدير **الانتقال السلس** بين الصفحات:

1. تضبط `opacity` لطبقة التلاشي (`#fade-overlay`) إلى `1` (كاملة التعتيم).
2. بعد تأخير (600 ميلي ثانية افتراضيًا)، توجّه المتصفح إلى الرابط الجديد.

هذا يخلق تأثير تلاشي أنيق إلى اللون الأسود قبل الانتقال إلى الصفحة التالية.

## تخصيص اللوحات

كل لوحة تتبع نظام الألوان الخاص بفرعها:
| الفرع | اللون | النص |
|-------|-------|------|
| العلوم الطبيعية | `#00d4ff` (سيان) | العلوم الطبيعية |
| العلوم الاجتماعية | `#9d00ff` (أرجواني) | العلوم الاجتماعية |
| العلوم الشكلية | `#00ff87` (أخضر) | العلوم الشكلية |

هذا يساعد المستخدم على تمييز الفروع بسهولة وتعزيز الهوية البصرية لكل مجال.
