# index.html — المدخل الرئيسي

## موقعه في المشروع

هذه هي **الصفحة الرئيسية** التي تفتح عند بدء التجربة. المستخدم يرى كهفًا بلوريًا طويلاً وثلاثة فروع جانبية متوهجة تمثل مجالات المعرفة الرئيسية.

---

## مكونات الصفحة

### 1. هيكل HTML

```html
<!DOCTYPE html>
<html lang="ar">
```

- اللغة العربية (`lang="ar"`) تدعم اتجاه الكتابة من اليمين لليسار.
- يستورد مكتبة Three.js من CDN:
  ```html
  <script src="https://cdnjs.cloudflare.com/ajax/libs/three.js/r128/three.min.js"></script>
  ```
- يربط ملف التنسيقات `styles/main.css`.
- يستورد ملف JavaScript الرئيسي:
  ```html
  <script type="module" src="js/main.js"></script>
  ```
  - `type="module"` يسمح باستخدام `import` و `export` داخل الكود.

### 2. عناصر HTML

- **`#fade-overlay`** — طبقة سوداء شفافة تُستخدم للتلاشي (fade) عند الانتقال بين الصفحات. تمنع pointer-events حتى لا تعيق التفاعل.
- **`#orientation-screen`** — شاشة تظهر في الوضع العمودي (portrait) على الجوال، تخبر المستخدم بتدوير الجهاز أفقيًا للاستمتاع بالتجربة.

### 3. تدفق العمل (باختصار)

`main.js` هو المسؤول عن كل شيء هنا:
1. إنشاء مشهد Three.js بكهف بلوري طويل
2. إضافة 3 فروع جانبية مع لوحات إرشادية متوهجة
3. السماح للمستخدم بالتحرك باللمس (السحب) أو عجلة الفأرة
4. عند النقر على لوحة فرع → انتقال متدرج إلى `sections.html?domain=...`

---

## ما الذي يجعل هذه الصفحة مميزة؟

- تستخدم **وحدات ES (Modules)** لتنظيم الكود النظيف والقابل للصيانة
- توفر تجربة غامرة (immersive) فور تحميل الصفحة
- تتعامل مع شاشة اللمس والماوس بسلاسة
- تصميم متجاوب (responsive) مع الأجهزة المحمولة
