### شفافية اللوحات (Fade)

| المنطقة | السلوك |
|----------|--------|
| `relativeZ > -400` | تتلاشى عند الاقتراب (على مدى 500 وحدة) |
| `-1600 < relativeZ < -400` | كاملة الشفافية (`opacity = 1`) |
| `relativeZ < -1600` | تتلاشى عند الابتعاد (على مدى 800 وحدة) |

**عدّل القيم:** `-400`, `-1600`, `500`, `800` في `js/posts-main.js:119-125`.

### دوران اللوحات

| المنطقة | السلوك |
|----------|--------|
| `relativeZ >= -50` | اللوحة تواجه الكاميرا مباشرة (زاوية 0) |
| `-450 < relativeZ < -50` | دوران تدريجي (على مدى 400 وحدة) |
| `relativeZ <= -450` | الزاوية الأصلية (`±0.32` راديان ≈ 18°) |

**لتغيير زاوية الدوران:** غيّر `0.32` في `js/posts-main.js:127`.

## 🧩 16. التشوه البلوري (Crystal) - ترجمة القيم

```
Math.sin(z * 0.03) * 40
│      │         │
│      │         └── أقصى إزاحة للخارج (40 وحدة)
│      └── تردد التموج على طول المحور Z
└── دالة الجيب لتموج ناعم

Math.cos(angle * 6 + z * 0.01) * 25
│      │              │         │
│      │              │         └── أقصى إزاحة (25 وحدة)
│      │              └── تموج بطيء على الطول
│      └── تموج سريع حول المحيط (6 تموجات لكل دورة)
└── دالة جيب التمام
```
